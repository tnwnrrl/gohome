## 스토리 개요

우주기지에서 동면에서 깨어난 플레이어들은 AI 도로시의 안내로 임무를 수행한다.
다른 행성계의 기지라고 생각했지만, 사실 이곳은 **달 기지**였다는 것을 알게 되고,
우리도 사실은 인간이 아닌 **달 기지에 버려져 폐기된 안드로이드**라는 사실을 알게 된다.

도로시도 사실 우리와 같은 안드로이드이고, 우리를 만든 창조자가 불쌍히 여겨
**지구로 돌아올 수 있는 마지막 임무**를 받게 된다.

하지만 도로시는 빠져나올 수 없는 상황.
우리는 기지에 남은 마지막 귀환캡슐에 탑승하게 되고,
도로시를 남긴 죄책감에 지구로 도착하지만...

**마지막에 우주선에서 익숙한 도로시의 목소리가 들리면서 끝난다.**

```

│  ACT 1: 각성                                            │

│  ACT 2: 의문                                            │
│  "이상한 점들 발견... 뭔가 이상해"                       │

├─────────────────────────────────────────────────────────┤
│  ACT 3: 진실 (반전 연속)                                │
│  → 반전①: 시크릿 코드 발견 → "여기 달이야"              │
│  → 반전②: 창조주 화상편지 → "우린 안드로이드야"         │
│  → 반전③: 도로시 실제 모습 발견 → "너도 우리랑 같았구나" │
├─────────────────────────────────────────────────────────┤
│  ACT 4: 선택 / 이별                                     │
│  "창조자의 마지막 선물 - 귀환 임무"                      │
│  → 캡슐은 하나, 도로시는 남아야 함                       │

│  ACT 5: 귀환 (엔딩 연출)                                │

└─────────────────────────────────────────────────────────┘
```

## 세계관 설정

### 배경
- **장소**: 달 기지 (플레이어는 처음에 다른 행성계라고 착각)
- **시대**: 미래, 안드로이드 기술이 발달한 시대
- **상황**: 달 기지 임무 완료 후, 복귀 비용 절감을 위해 안드로이드들을 달에 폐기

### 창조자
- 과학자
- 안드로이드들을 불쌍히 여겨 마지막 귀환 기회를 설계해둠

### 정체 발견 트리거

#### 플레이어: 점점 의심 축적 (C타입)
- **"인간 출입 불가 구역"**을 무사히 통과하면서 의문 발생
- 방사능 구역 / 저산소 구간 / 극저온/고온 구역 등
- "왜 안 춥지?" → "왜 숨 안 쉬어도 되지?" → "우리... 설마?"
- **시크릿 코드 발견 → 입력 → 진실 해금**

#### 도로시: 코드 입력 후 함께 깨달음 (B타입)
- 처음엔 플레이어를 인간으로 알고 있음
- 시크릿 코드 입력 순간 시스템 데이터 접근
- 플레이어와 함께 진실을 알게 됨 → 더 큰 감정적 유대
- **"속았다"가 아닌 "학습/성장 과정"으로 이해**

#### 창조자의 의도
- 악의로 숨긴 것이 아님
- 안드로이드들이 스스로 진실을 발견하고 받아들일 준비가 될 때까지 기다림
- 보호와 교육의 마음으로 설계된 "성장 과정"

#### 메타 연출: 느낄 수 없는 위험
```
스토리        : 극저온/고온 구역 → 인간은 위험
안드로이드    : 온도 감각 없음 → 영향 없음
실제 방탈출  : 온도 구현 불가 → 체감 없음
플레이어     : "이상하게 안 춥네/안 덥네..."
→ 게임의 물리적 한계가 오히려 스토리 복선이 됨!
```

| 구역 | 시각 연출 | 청각 연출 | 도로시 대사 |
|------|-----------|-----------|-------------|
| **극저온실** | 파란 조명, 서리 장식, "-40°C" 표시 | 냉각 윙윙 소리 | "여긴 좀 추울 거예요~ 아, 근데 괜찮으시죠?" |
| **고온 구역** | 빨간 조명, 열기 이펙트, "WARNING" | 경고음, 기계 굉음 | "조심하세요, 여기 뜨거... 어? 괜찮네요?" |
| **방사능 구역** | 노란 경고등, 방사능 표지 | 가이거 계수기 소리 | "인간은 30초도 못 버티는데... 음, 그냥 가세요!" |
| **저산소 구간** | "O₂ 12%" 표시, 붉은 조명 | 경고 알람, "산소 부족" 방송 | "숨 참으세요! ...어? 안 참아도 되시네요?" |

### 도로시 백업 복선
- 플레이어가 게임 중 수행한 **"어떤 행동"**으로 인해 도로시가 백업됨
- 의도치 않은 행동이 나중에 의미있게 드러나는 구조
- 후보:
  - 초반 "기지 로그 백업" 임무로 위장
  - 귀환캡슐-기지 연결 과정에서 자동 동기화
  - 도로시가 "이건 챙겨가"라고 준 물건 (메모리 칩)

## 오프닝 연출

### 캡슐 구성

| 항목 | 설정 |
|------|------|
| **인원** | 최대 5명 |
| **구조** | 1인 1캡슐 |

| **화면** | 캡슐 앞 대형 TV |

```
캡슐 배치도 (위에서 본 모습)

     [1] [2] [3] [4] [5]    ← 동면 캡슐 (5개)
     ───────────────────
          [ TV 화면 ]       ← 공용 대형 화면
```

### 동면 해제 시퀀스

```
┌─────────────────────────────────────────────────────────┐
│  STEP 1: 입장                                           │
│  플레이어들 각자 캡슐에 입장 → 문 닫힘                   │
├─────────────────────────────────────────────────────────┤
│  STEP 2: 동면 시작 (연출)                               │
│  암전 → 잠시 대기                                       │
├─────────────────────────────────────────────────────────┤
│  STEP 3: 동면 해제                                      │
│  • 시스템 부팅음                                        │
│  • 도로시 음성 시작                                     │
│  • 연무기/가습기 → 캡슐 내부 안개 효과                  │
│  • 공압장치 → "쉬익" 해제음                            │
├─────────────────────────────────────────────────────────┤
│  STEP 4: 브리핑                                         │
│  TV 화면 ON → 미션 브리핑 영상                          │
├─────────────────────────────────────────────────────────┤
│  STEP 5: 게임 시작                                      │
│  캡슐 도어 오픈 → 탐색 시작                             │
└─────────────────────────────────────────────────────────┘
```

### 연출 장치

| 장치 | 용도 | 비고 |
|------|------|------|
| **연무기/가습기** | 동면 해제 안개 효과 | 캡슐 내부 또는 하단 |
| **공압장치** | 캡슐 문 개방 시 "쉬익" 소리 | 연출용 |
| **조명** | 암전 → 점멸 → 점등 | 분위기 전환 |
| **스피커** | 도로시 음성 + 효과음 | 기지 방송 시스템 |

## 임무 및 퍼즐 구조

### 표면적 임무
**"기지 시스템 점검 및 복구"**
- 인간에겐 위험/불가능 → 안드로이드엔 단순 작업
- 난이도: 쉬움~중간 (초보자 친화)
- 복선: "왜 이게 우리한텐 쉽지?" → 정체 의심

### 안드로이드라서 가능한 단순 작업들

| 작업 | 인간이면? | 안드로이드면? | 퍼즐 예시 |
|------|-----------|---------------|-----------|
| 방사능 구역 진입 | 방호복, 시간 제한 | 그냥 들어감 | 경고 무시하고 통과 |
| 저산소 구간 통과 | 숨 참기, 빠른 이동 | 호흡 불필요 | 여유롭게 작업 |
| 고온/저온 환경 | 체온 문제 | 영향 없음 | 냉각실/가열실 작업 |
| 정밀 코드 입력 | 외우기 어려움 | 한 번 보면 기억 | 긴 숫자열 입력 |
| 무거운 패널 이동 | 힘들음 | 가볍게 | 장치 재배치 |
| 전기 회로 직접 연결 | 감전 위험 | 문제없음 | 맨손 회로 작업 |

### 퍼즐 흐름 (예시)

```
ACT 1-2: 점검 임무 (쉬움)
├── "전원 패널 재가동" - 스위치 조합
├── "통신 모듈 연결" - 케이블 매칭
└── "냉각 시스템 점검" - 저온 구역 진입 ← 첫 번째 이상함

ACT 2-3: 복구 임무 (중간)
├── "손상 구역 수리" - 방사능 경고 무시하고 진입
├── "데이터 백업" - 긴 코드 입력 (도로시 백업 복선!)
└── "저산소 구간 통과" - 숨 안 쉬고 작업 ← 결정적 의문

ACT 4: 귀환 준비
└── "캡슐 발사 시퀀스" - 최종 퍼즐
```

### 도로시의 복선 대사들

```
초반: "여러분 정말 튼튼하시네요~ (작게) 당연하지만..."

중반: "보통 사람들은 여기서 힘들어하는데...
       아, 여러분은 특별 훈련 받으셨구나! 네, 그거예요."

후반: "솔직히 말할게요.
       여러분이 이걸 느끼지 못하는 이유는..."
```

## 엔딩 연출

### 1. 도로시 발견
- 수리캡슐 직접 보게 됨
- 도로시 모습 공개 (안드로이드 형태)
- 움직일 수 없는 상태

### 2. 이별 (유쾌하게 숨기며)
슬픔을 유쾌함으로 감추는 도로시

```
"뭐 그렇게 쳐다봐요~ 예쁘죠? 저?"
"울지 말아요, 저 원래 눈물 싫어해요."
"빨리 가요. 지구 햇살 대신 쬐고 와요."
"...가끔 생각은 해줘요, 알았죠?"
```

### 3. 귀환 캡슐 탑승
- 별도 귀환 캡슐 (오프닝 동면 캡슐과 다름)
- 플레이어 전원 탑승
- 발사 시퀀스

### 4. 엔딩 (수미상관)
