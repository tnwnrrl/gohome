# GO HOME 4D 하이브리드 모션 플랫폼 상세 설계

> 5인 탑승 귀환 캡슐용 모션 시뮬레이터

---

## 1. 설계 개요

### 1.1 요구 사양

| 항목 | 사양 |
|------|------|
| **탑승 인원** | 5명 |
| **최대 하중** | 400kg (인체 350kg + 플랫폼 구조물 50kg) |
| **운동 범위** | Pitch 10°, Roll 10°, Heave ±50mm |
| **액추에이터** | 4개 (사각 배치) |
| **자유도** | 4DOF (Pitch, Roll, Heave 독립 + 복합) |
| **체감 보조** | 진동 모터, 저음 우퍼, 송풍기 |

### 1.2 설계 철학

```
실제 모션 (5~10°) + 체감 보조 장치 = 몰입감 극대화
└── 작은 물리적 움직임으로 안전성 확보
└── 진동/음향/바람으로 체감 증폭
```

### 1.3 좌표계 정의

```
        +Y (전방)
         ↑
         │
         │
 -X ←────┼────→ +X (우측)
         │
         │
         ↓
        -Y (후방)

Pitch: X축 회전 (앞뒤 기울임)
Roll:  Y축 회전 (좌우 기울임)
Heave: Z축 이동 (상하 움직임) ← 4DOF 추가
```

## 3. 액추에이터 배치 및 사양

### 3.1 4점 사각 배치 (4DOF)

```
액추에이터 배치도 (위에서 본 모습)
───────────────────────────────────────

              [전방]
                ↑
   ┌────────────────────────────┐
   │                            │
   │  ● ACT1            ACT2 ●  │
   │  (전방좌)          (전방우)│
   │                            │
   │           중심             │
   │            ╋              │
   │                            │
   │  ● ACT3            ACT4 ●  │
   │  (후방좌)          (후방우)│
   │                            │
   └────────────────────────────┘
                ↓
              [후방]

   배치 형태: 직사각형
   전후 간격: 1600mm
   좌우 간격: 1400mm
   중심점: 플랫폼 무게중심
```

### 3.2 배치 좌표

| 액추에이터 | X (mm) | Y (mm) | 역할 |
|------------|--------|--------|------|
| ACT 1 | -700 | +800 | 전방좌 |
| ACT 2 | +700 | +800 | 전방우 |
| ACT 3 | -700 | -800 | 후방좌 |
| ACT 4 | +700 | -800 | 후방우 |

### 3.3 액추에이터 사양

#### 3.3.1 추천 제품

**제품명: 전동 리니어 액추에이터 (산업용)**

| 항목 | 사양 |
|------|------|
| **타입** | 전동 리니어 액추에이터 |
| **스트로크** | 200mm |
| **정격 하중** | 200kg (Push/Pull) |
| **최대 하중** | 300kg |
| **속도** | 20~50mm/s (조절 가능) |
| **전압** | DC 24V 또는 AC 220V |
| **IP 등급** | IP65 |
| **피드백** | 포텐셔미터 내장 |

#### 3.3.2 하중 계산

```
총 하중 분석
───────────────────────────────────────

정적 하중:
├── 탑승자 5명: 350kg (70kg × 5)
├── 상부 프레임: 약 80kg
├── 상판 + 좌석: 약 50kg
└── 체감 장치: 약 20kg
    ────────────────
    합계: 500kg

동적 하중 (가속도 고려):
├── 동적 계수: 1.5 (안전율)
└── 설계 하중: 500kg × 1.5 = 750kg

액추에이터당 하중 (4점 지지):
├── 균등 분배: 750kg ÷ 4 = 187.5kg
└── 불균등 시 최대: 약 250kg (한쪽 기울임시)

→ 300kg급 액추에이터 선정 (여유율 확보)
→ 4점 지지로 개별 부담 감소, 안정성 향상
```

#### 3.3.3 스트로크 계산

```
각도-스트로크 관계 (삼각 배치)
───────────────────────────────────────

피벗 중심에서 액추에이터까지 거리: R = 900mm

최대 각도 10° 기울임 시:
Δh = R × sin(10°) = 900 × 0.1736 = 156mm

필요 스트로크:
├── 상승: +78mm (5° 상승)
├── 하강: -78mm (5° 하강)
└── 총 스트로크: 156mm

→ 200mm 스트로크 액추에이터 선정 (여유 확보)
```

```

| Roll 우→   | 수축       | 신장       | 수축       | 신장       |
| Heave 상↑  | 신장       | 신장       | 신장       | 신장       |
| Heave 하↓  | 수축       | 수축       | 수축       | 수축       |
|------------|------------|------------|------------|------------|
| 복합: 앞좌 | 신장       | 중립       | 중립       | 수축       |

| 복합: 뒤좌 | 중립       | 수축       | 신장       | 중립       |
| 복합: 뒤우 | 수축       | 중립       | 중립       | 신장       |

4DOF 장점:
├── Heave(상하): 발사/착륙 충격 표현 가능
├── 독립 제어: 4개 모서리 개별 제어
└── 복합 동작: 더 자연스러운 움직임
```

## 4. 조인트 설계

### 4.1 조인트 유형 선택

#### 4.1.1 비교 분석

| 유형 | 장점 | 단점 | 적합도 |
|------|------|------|--------|
| **로드엔드 베어링** | 가격 저렴, 구매 용이 | 각도 제한(±15°), 마모 | **추천** |
| 볼조인트 | 넓은 각도 범위 | 고가, 정밀 가공 필요 | 중간 |
| 유니버설 조인트 | 고하중, 내구성 | 매우 고가, 오버스펙 | 부적합 |

#### 4.1.2 선정: 로드엔드 베어링

**제품명: 로드엔드 베어링 (Rod End Bearing / Heim Joint)**

| 항목 | 상부 연결용 | 하부 연결용 |
|------|-------------|-------------|
| **규격** | SA20T/K (암나사) | SA20T/K (암나사) |
| **내경** | 20mm | 20mm |
| **정격 하중** | 40kN (약 4000kg) | 40kN |
| **각도 범위** | ±14° | ±14° |
| **재질** | 크롬강 + 청동 슬리브 | 크롬강 + 청동 슬리브 |

### 7.2 세부 조립 지침

#### 7.2.1 1단계: 프레임 제작

```
하부 베이스 용접 순서
───────────────────────────────────────

   1. 종빔 2개 평행 배치 (2400mm 간격 유지)

      │<─── 2200mm ───>│
      ╔═══════════════╗
      ║               ║
      ║               ║
      ╚═══════════════╝

   2. 횡빔 용접 (직각 확인 필수!)

      ┌───────────────┐
      │               │ ← 직각자로 확인
      │               │
      └───────────────┘

   3. 마운트 플레이트 용접

   ※ 주의: 용접 순서를 지키지 않으면 열변형 발생
   ※ 대각선 순서로 용접하여 변형 최소화
```

#### 7.2.2 3단계: 액추에이터 설치

```
액추에이터 장착 순서
───────────────────────────────────────

1. 하부 로드엔드 베어링 체결
   - 액추에이터 본체 끝단에 로드엔드 나사 체결
   - 록타이트 도포 (중강도)

2. 하부 클레비스에 액추에이터 삽입
   - 핀 홀 정렬
   - 핀 삽입 + 와셔 + 스냅링

3. 상부 로드엔드 베어링 체결
   - 액추에이터 로드 끝단에 로드엔드 나사 체결

4. 액추에이터 길이 조정
   - 3개 동시에 중간 위치로 조정
   - 스트로크 100mm 위치 (총 200mm 중)
```

#### 7.2.3 4단계: 플랫폼 연결

```
상부 플랫폼 연결 작업
───────────────────────────────────────

※ 중요: 상부 플랫폼은 무거우므로 반드시 호이스트 사용!

1. 호이스트로 플랫폼 들어올리기
   - 4점 슬링 사용
   - 수평 유지

2. 액추에이터 상부 조인트 위치 맞춤
   - 천천히 하강
   - 3개 조인트 동시에 맞추기

3. 핀 삽입
   - 한 명은 플랫폼 잡고
   - 다른 한 명이 핀 삽입

4. 스냅링 체결

5. 호이스트 해제 후 수평 확인
   - 수평계로 확인
   - 필요시 액추에이터 길이 미세 조정
```

## 11. 제어 소프트웨어 개요

### 11.1 시퀀스 데이터 포맷

```cpp
// Arduino 시퀀스 예시 (4DOF)
struct MotionFrame {
    uint32_t timestamp_ms;  // 타임스탬프
    int16_t act1_pos;       // ACT1 위치 - 전방좌 (0-1000)
    int16_t act2_pos;       // ACT2 위치 - 전방우 (0-1000)
    int16_t act3_pos;       // ACT3 위치 - 후방좌 (0-1000)
    int16_t act4_pos;       // ACT4 위치 - 후방우 (0-1000)
    uint8_t vibration;      // 진동 강도 (0-255)
    uint8_t fan_speed;      // 팬 속도 (0-255)
    uint8_t bass_level;     // 우퍼 레벨 (0-255)
};

// 4DOF 동작 헬퍼 함수
void setPitch(int angle);   // 앞뒤 기울임 (-10° ~ +10°)
void setRoll(int angle);    // 좌우 기울임 (-10° ~ +10°)
void setHeave(int mm);      // 상하 이동 (-50mm ~ +50mm)
```

### 11.2 발사 시퀀스 타임라인 (4DOF)

```
발사 시퀀스 (60초) - Heave 동작 추가
═══════════════════════════════════════════════════════════════

시간(초)    모션                      진동   팬   우퍼
─────────────────────────────────────────────────────────────
0-10      정지                      0      0    긴장감 저음
10-13     카운트다운                 미세    0    카운트 사운드
13-15     발사! Pitch뒤10°+Heave↑   MAX    50%  폭발음
          └ Heave 위로 쿵! (G포스 체감)
15-25     대기권 Roll+Heave 흔들림   HIGH   80%  엔진음
          └ 불규칙 상하+좌우 진동
25-40     우주비행 미세Roll          LOW    20%  저주파
40-50     지구진입 흔들림+Heave      HIGH   MAX  마찰음
          └ 전신 흔들림
50-55     감속 Heave↓               MED    50%  감속음
          └ 서서히 하강 느낌
55-58     착륙충격 Pitch앞+Heave↓↓  MAX    0    쿵!
          └ 강한 하방 충격!
58-60     정지                      0      0    침묵
60-       엔딩 음성                  0      0    "안녕하세요~"

4DOF Heave 효과:
├── 발사: 위로 솟구치는 G포스
├── 대기권: 불규칙 상하 진동
├── 착륙: 땅에 닿는 충격감
└── 3DOF 대비 체감 2배 이상 향상
```
